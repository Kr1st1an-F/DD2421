## Project Structure

| Directory | Description |
|-----------|-------------|
| **`Lab 1/`** | **Decision Trees**: Builds and evaluates classification/regression trees. Covers **entropy/Gini impurity**, **pruning strategies**, and **hyperparameter tuning** (max_depth, min_samples_split). Applied to datasets like Iris with accuracy, precision, and tree visualization. |
| **`Lab 2/`** | **Support Vector Machines (SVM)**: Implements **linear, RBF, and polynomial kernels**. Explores **hard/soft margins**, **C regularization**, and **SMO-style optimization**. Includes **cross-validation** for kernel selection and **decision boundary plots**. |
| **`Lab 3/`** | **Bayesian Learning & Boosting**|
| **`Challenge/`** | **The final Exam**|
